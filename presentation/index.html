HTML — HyperText Markup Language

  • tags
    • eg: <title>My Title</title>
    • enclosed in angle brackets and usually consist of a start tag and an end tag
    • can be nested
  • tag attributes
    • eg: <a href="http://www.google.com/">a link to google</a>
    • name/value pairs within the start tag giving extra information about the element
    • values may be enclosed in '' or ""
  • content
    • the text or other element between the start and end tag

HTTP - HyperText Transfer Protocol

  • the protocol for transferring html and other world wide web data between computers
  • a request-response protocol, usually stateless
